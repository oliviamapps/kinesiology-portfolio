Python Analysis Tool

V̇O2 Kinetics and Gas Exchange Analysis Using Python

Includes programmatically generated plots for FECO2, VCO2, and identification of the Gas Exchange Threshold (GET), and Respiratory Compensation Point (RCP)

Analytical features:

  • dynamic plot annotation: integrated vertical reference lines directly into plots, removing the need for manual annotation and improving reproducibility
  • applied course-based physiological principles to inform threshold detection methods, ensuring analytical validity
  • reproducible code by changing the input file path for a new dataset
