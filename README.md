# Olivia Mappalakayil | Kinesiology Portfolio

A public-facing academic portfolio built for my fourth-year Kinesiology course at the University of Calgary. This site showcases my applied computational work, data visualizations, and technical projects related to exercise physiology and human movements.

Live site: (https://oliviamapps.github.io/kinesiology-portfolio/index.html#)

## Tools Used 
HTML: Page structure and content 
CSS: Styling, layout, navigation
Python: Data analysis and programmatic visualization
Excel: Kinesiology-related calculation system
Scratch: Logic-based interactive maze

##How to Navigate the Site 
Use the navbar at the top of every page to move between sections. The Projects dropdown contains all technical work.
